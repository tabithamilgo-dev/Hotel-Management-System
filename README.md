# Hotel-Management-System
This project is a Hotel Management System built with Django as the backend framework and HTML, CSS, and Bootstrap for the frontend. It allows hotel staff to manage rooms, bookings, customer details, and payments in a user-friendly interface.

# Features
- User registration and login
- Room booking and management
- Customer management
- Payment tracking

# Tech Stack
- Python 
- Django 
- SQLite 
- HTML, CSS, Bootstrap

# How to Run
1. Clone the repository
2. Create a virtual environment: `python -m venv env`
3. Activate the environment
4. Install requirements: `pip install -r requirements.txt`
5. Run migrations: `python manage.py migrate`
6. Start the server: `python manage.py runserver`
